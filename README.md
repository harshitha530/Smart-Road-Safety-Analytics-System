# Smart Road Safety Analytics & Risk Assessment System

## Project Overview

The Smart Road Safety Analytics & Risk Assessment System is a data-driven project designed to analyze road accident data and identify factors contributing to accidents. The system uses machine learning techniques to predict risk levels and provides interactive visualizations through a Tableau dashboard.

## Objectives

* Analyze road accident patterns and trends.
* Identify key factors affecting road safety.
* Predict accident risk scores using machine learning.
* Visualize insights through an interactive dashboard.
* Support data-driven decision making for road safety improvement.
